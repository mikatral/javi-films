# 🎬 JaviFilms

Aplicação web full stack desenvolvida para o trabalho final da disciplina **Linguagem de Programação para Internet 1** da UNIUBE.  
Permite o **cadastro, edição, listagem e exclusão de filmes**, incluindo título, categoria e imagem via URL.

---

## 🧪 Tecnologias Utilizadas

### 🔹 Front-end:
- React
- React Router DOM
- Axios
- CSS puro

### 🔹 Back-end:
- Node.js
- Express
- MongoDB Atlas
- Mongoose
- CORS

---

## 🚀 Funcionalidades

- [x] Cadastro de filmes com título, categoria e imagem
- [x] Edição de filmes já cadastrados
- [x] Listagem de todos os filmes
- [x] Exclusão com confirmação
- [x] Validação de campos obrigatórios
- [x] Imagem com tratamento de erro (fallback)
